    What if we want \LaTeX \: in our blog posts?

        draft: true
        math: true
        ---

        # Hello, world!

        This is my first blog post. In my post, I can do things like:

        - Make lists
        - _Emphasize_ text
        - Make text **bold**
        - Show off some inline \LaTeX
